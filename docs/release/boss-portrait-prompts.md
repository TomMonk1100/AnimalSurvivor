# Boss portrait prompts

These two square bitmap portraits are presentation-only UI assets for the
boss-health panel. They do not enter simulation state, replay inputs, hashes,
or balance. They were generated with the built-in image-generation tool on
2026-07-12 and downscaled to 768x768 RGB PNGs for the runtime asset budget.
