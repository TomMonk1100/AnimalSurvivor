# Gate 0 Image Prompts

These are project-bound concept prompts for the built-in image-generation path.
They produce reference boards, not final runtime models.

## Shared constraints

- Original low-poly 3D game concept art; no resemblance to an existing franchise.
- Orthographic three-quarter gameplay camera and isolated character turntable
  views when requested.
- Cute heroic animal at the start; increasingly strange, mutated, and powerful at
  the end; never gory or frightening.
- Faceted geometry, economical materials, browser-game-feasible forms.
- No text, logos, interface labels, trademarks, or watermarks.
- Face and original species remain recognizable in every stage.

## Board A — Storybook Wildguard

```text
Use case: stylized-concept
Asset type: low-poly 3D game visual-direction board and character progression
Primary request: show one original small heroic forest animal in three stages: clean natural beginner, mid-run champion with three visible animal-trait attachments, and end-run mythic chimera whose attachments have fused into a coherent powerful silhouette
Scene/backdrop: warm cream studio board with small inset of a sunny meadow combat arena
Subject: rounded appealing proportions, expressive face, beetle-shell back armor, compact porcupine quills, and an electric tail evolving into polished natural regalia
Style/medium: polished low-poly 3D game render, faceted surfaces, toy-like but not plastic, feasible real-time geometry
Composition/framing: landscape concept board; three full-body progression stages plus one small three-quarter gameplay view
Lighting/mood: warm storybook daylight, brave and optimistic
Color palette: coral, teal, warm gold, cream, restrained forest green
Constraints: same recognizable animal identity in every stage; no text; no logo; no gore; no weapons held in hands; no watermark
Avoid: photoreal fur, high-detail textures, generic mobile-game UI, visual clutter
```

## Stat-only control edit
