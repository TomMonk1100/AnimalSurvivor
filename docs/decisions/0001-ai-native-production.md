# ADR 0001: AI-Native Production

**Status:** Accepted  
**Date:** 2026-07-10

## Decision

AI agents perform all production work. The owner supplies creative direction and
approvals. Available subscription usage is the operative budget.

## Consequences

- Work must be resumable from repository artifacts.
- Agent tasks require strict directory boundaries and acceptance tests.
- No runtime AI dependency or paid API is permitted.
- Model usage is prioritized toward integrated code and verification.
- Human preference remains a separate kind of evidence and cannot be simulated.
