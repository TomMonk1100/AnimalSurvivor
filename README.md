# Animal Survivor

Pre-production workspace for a zero-cash, web-first, low-poly 3D animal
bullet-heaven game in which the player's build is physically visible on the hero.

The current greenlight recommendation, product definition, validation gates,
technical architecture, and agent-swarm execution plan are in
[`docs/greenlight-and-swarm-plan.md`](docs/greenlight-and-swarm-plan.md).

The project is AI-produced with the owner acting as creative director. Usage-aware
checkpoint and continuation rules are in
[`docs/ai-native-operating-model.md`](docs/ai-native-operating-model.md).

No production code has been started. The first implementation milestone is a
disposable PlayCanvas performance-and-visual prototype using audited CC0 assets,
not the full game.
