# ADR 0002: Proceed to Gate 1 Before Human Concept Interviews

**Status:** Accepted with risk  
**Date:** 2026-07-10

## Context

The Gate 0 artifact package is complete, but ten human concept interviews have not
occurred. This is a zero-cash serious hobby project whose primary cost is AI usage.

## Decision

Proceed to the small Gate 1 technical toy while retaining the missing human
validation as an explicit risk. Move the hard human checkpoint to Gate 2, when a
real ten-minute web build exists.

## Consequences

- Gate 0 is not described as human-validated.
- Gate 1 remains disposable and tightly scoped.
- No roster/content expansion occurs before playable human feedback.
- Synthetic agent critique is never labeled player testing.
